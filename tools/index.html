Tools

Tools such as text analyzers and programming language interpreters. Some use the library p5.js.

List of Tools:

  • Conjugator - Conjugate and deconjugate Spanish present tense verbs
  • Big Text - Show big text with custom fonts and colors
  • Encoder - Create 'secret' URLs
  • Maze Maker - Create customized mazes
  • Converter - Convert between different units and scales
  • Color Classifier - Type in a hex code to get the name of the color
  • Markov Word Generator - Generate fake words based on real text
  • Text Analyzer - Analyze text for statistics and word and character frequencies
  • Programming - Esoteric programming languages
  • Whoami - The information websites can find out about you
