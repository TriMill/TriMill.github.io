Color of the day

Select date:


The color is randomly generated using the date as a seed, so each day will always give the same number, no matter when it is accessed or what browser you use. There are over 16 million possible colors, so it is unlikely that the same color will ever appear twice (although it could have the same name). The names are usually accurate, but it's not hard to find completely wrong ones (ex. 08/25/2010). Try entering your birthdate or another important date and see what color you get!
