For a long time, 0 was not considered a number. After all, how can the lack of anything be a thing in and of itself? Positional systems (like decimal and binary) need a 0 symbol, otherwise, for example, the numbers 203 and 23 would be written the same. Many systems started out by using a space, but this still leaves room for confusion.

The symbol used for 0 likely originated in India along with the base-10 system used in most of the world. Positional systems are much more efficient for calculations than other systems used at the time, like Roman numerals. As such, Arabic numerals were adopted quite quickly. For a while after, though, 0 was considered just a placeholder symbol and not an actual number.

When people discuss weird numbers, they often talk about irrationals (π, e, ϕ), infinities (ℵ0, ω), quaternions and octonions, etc. They often miss perhaps the strangest number of all, 0. Every single real number, every complex, every quaternion and octonion, every ordinal and cardinal, even every surreal number has a reciprocal. 0 does not. Many people erroneously believe the reciprocal of 0 is "infinity", which if true would likely also make 2/0 "infinity". If 1/0 = 2/0, multiplying both sides by 0 gives 1 = 2, which is clearly false.

Information

0 is the first natural number (integers greater than or equal to 0).
It is the additive identity, so adding 0 to a number does not change it: a + 0 = a
Multiplying anything by 0 yields 0: a · 0 = 0, and as a result, division by 0 is impossible (no number times 0 will give the numerator).
Any non-0 number raised to the 0th power is 1: a0 = 1. 0 to the power of 0 is undefined.
0 is neither prime (since any number times 0 is 0, it has all numbers for factors) nor composite (since it cannot be written as a product of primes without using 0). Since it is divisible by all numbers except itself, it is even.
0 is the 0th term of the Fibbonacci sequence.
0 is written "0" in all natural number bases.
